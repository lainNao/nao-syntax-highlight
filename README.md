# nao-syntax-highlight

VSCode extension for `.nao` extension file (super simple markdown for me).

```
//...
■...
＜...＞
```

## install from local
1. clone
    - `git clone https://github.com/lainNao/nao-syntax-highlight`
2. build
    - `npx vsce package`
3. install
    - a

## development
- `F5` to open debugging window
